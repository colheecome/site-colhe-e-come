Stripe — Pagamento Online

Configure os links de pagamento. Crie cada link em dashboard.stripe.com → Payment Links e cole abaixo.

Link de checkout geral

Link usado quando o cliente finaliza o carrinho com múltiplos produtos. Crie um Payment Link genérico ou use o link de uma página de pagamento personalizada.

Links por produto

Cada produto pode ter seu próprio Payment Link do Stripe. Edite o produto na aba Produtos para inserir o link individual.

Como criar um Payment Link

Passo a passo:

  1. Acesse dashboard.stripe.com e faça login
  2. Vá em Payment Links → + Novo link
  3. Crie um produto com o nome e preço, clique em Criar link
  4. Copie a URL gerada (ex: https://buy.stripe.com/abc123) e cole no campo do produto
  5. Para receber via Pix ou boleto: em Configurações → Métodos de pagamento, habilite Brasil

Novo Produto

Crie em dashboard.stripe.com → Payment Links

⚠ Use sempre URLs de imagem — não upload de arquivo

Imagens carregadas por arquivo ficam salvas apenas no seu navegador e não aparecem em outros dispositivos ou para os clientes. Use um serviço gratuito de hospedagem de imagens:

imgur.com imgbb.com cloudinary.com

Suba a foto em qualquer um desses sites → copie o link direto da imagem → cole no campo acima.
